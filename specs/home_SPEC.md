# Specification: Home Page (Landing)

## Overview
The **Home Page** is the marketing and entry landing page, introducing the platform's value proposition.

## UI/UX Principles
1.  **Hero Section**: Strong headline, value prop ("Master Your Engineering Skills"), and CTA ("Start Assessment").
2.  **Features Grid**: Highlight key features (Real-time scoring, Analytics, diverse subjects).
3.  **Visuals**: High-quality 3D or abstract tech illustrations.

## Functional Rules
1.  **Navigation**:
    -   **Navbar**: Links to Assessment, Profile, Login.
    -   **Hero CTA**: Links to `/assessment` (or Login if restricted).
2.  **Responsiveness**: Adjust layout for mobile (stack sections).

## Edge Cases
-   **Authenticated User**: CTA might say "Continue Assessment" instead of "Get Started".
